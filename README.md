

# 🏏 IPL Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on IPL ball-by-ball data to uncover insights about player performance, team strategies, and match dynamics.

---

## 🎯 Objectives
- Analyze top-performing batsmen and bowlers  
- Understand dismissal patterns  
- Identify highest scoring teams  
- Evaluate performance in different match phases (death overs)  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Streamlit  

---

## 📊 Key Analysis

### 🏏 Batting Analysis
- Top batsmen by runs  
- Strike rate analysis  
- Death over performance  

### 🎯 Bowling Analysis
- Top wicket-taking bowlers  
- Bowling strike rate (balls per wicket)  
- Economy vs wicket analysis  

### 📈 Match Insights
- Dismissal type distribution  
- Team-wise total runs  
- Over-wise scoring patterns  

---

## 🔍 Key Insights
- Mumbai Indians scored the highest total runs  
- “Caught” is the most common dismissal type  
- Death overs show aggressive batting behavior  
- Efficient bowlers take wickets frequently while maintaining control  

---

## 📊 Visualizations
- Bar charts (top players, teams)  
- Count plots (dismissal types)  
- Scatter plots (advanced analysis)  
- Bubble charts (multi-dimensional insights)
